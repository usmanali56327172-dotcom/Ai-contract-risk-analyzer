# Ai-contract-risk-analyzer
AI Contract Risk Analyzer is a full-stack SaaS platform that uses AI to detect risky clauses, missing terms, and compliance gaps in contracts. It offers AI-generated redlines, negotiation playbooks, RAG-based contract chat, OCR, e-signatures, Stripe billing, and OAuth login — built with NestJS, Next.js, PostgreSQL, Qdrant, and OpenAI.
